# Portfolio 
## Project description
My portfolio, built with NextJS, React and TailwindCSS.


## Installation

**Install dependencies**
```bash
npm install
```

**Run development server**
```bash
npm run dev
```

## Packages

- [Next.js](https://nextjs.org/docs)
- [Tailwindcss](https://tailwindcss.com/docs)
